# Vida Buddies Purees

Premium Aseptic Fruit Purees Website. Built with the "Best of the Best" technologies.

## Technology Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
- **Animations**: GSAP, Framer Motion
- **3D Graphics**: Three.js, React Three Fiber, Drei
- **Components**: Shadcn/UI pattern (Radix Primitives)
- **AI Assets**: "Nano Banana" (Google Gemini) generated images

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure
- `src/app`: App Router pages and layout.
- `src/components`: React components.
  - `sections`: Full page sections (Hero, Products).
  - `ui`: Reusable UI components (Buttons, etc.).
  - `canvas`: Three.js Canvas components.
- `src/lib`: Utilities.
- `public/images`: Generated assets.

## Aesthetic
- **Premium Dark Mode**: Deep black background (`#0a0a0a`) mixed with vibrant fruit colors.
- **Typography**: `Outfit` for display, `Inter` for body.
- **Visuals**: High-contrast, hyper-realistic liquid simulations and metallic packaging mocks.
